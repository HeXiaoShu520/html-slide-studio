# AI 生成 HTML 幻灯片

在 VS Code 中通过 Claude Code 插件直接生成幻灯片 HTML 代码。

## 使用方法

1. 在 Claude Code 对话中 `@` 引用本文件
2. **同时引用 `src/utils/slidePrompt.ts`**（包含完整生成规范）
3. 描述你想生成的内容，例如：

```
@AI生成提示.md @src/utils/slidePrompt.ts

生成关于 UDS 诊断协议的演示，包含：封面、协议简介、核心服务分类（卡片）、通信流程图、报文示例、应用场景
```

4. 将生成的代码保存为 `.html` 文件，用浏览器打开即可演示

## 快捷模板（复制后修改主题即可）

**技术方案**
> 生成技术方案介绍演示，包含：封面、背景与痛点、解决方案、核心功能（卡片）、技术架构、实施步骤、预期收益。主题：【填写你的主题】

**协议讲解**
> 生成技术协议或规范的教学演示，包含：封面、背景介绍、核心概念（卡片）、工作流程图、关键细节、应用场景。协议名称：【填写协议名】

**培训手册**
> 生成操作培训手册演示，包含：封面、培训目标、操作流程（步骤图）、关键操作说明、常见问题、注意事项。主题：【填写培训主题】

**产品发布**
> 生成产品发布介绍演示，包含：封面（产品名+slogan）、产品亮点（3-4个核心特性卡片）、功能详解、技术规格、应用场景。产品名：【填写产品名】

**项目汇报**
> 生成项目汇报演示，包含：封面、项目背景与目标、项目进展（时间线）、已完成成果、遇到的问题与解决方案、下一步计划。项目名：【填写项目名】
