# ATAG 示例网页

本目录包含使用 ATAG 方法论生成的示例网页。

## 已有示例

### 1. UDS 诊断服务 (uds-diagnostic.html)
- **主题**：UDS 统一诊断服务 (ISO 14229)
- **页面数**：7 页
- **配色**：电动蓝主题
- **内容**：
  - 封面页：UDS 简介
  - 协议简介：标准和应用
  - 架构层次：应用层、传输层、物理层
  - 核心服务：6个常用服务卡片
  - 通信流程：请求-响应模式
  - 报文示例：读取 VIN 码
  - 应用场景：维修、升级、测试

**特点**：
- ✅ 完整的 ATAG 设计系统
- ✅ 电动蓝配色方案
- ✅ Font Awesome 图标
- ✅ 流畅的翻页动画
- ✅ 响应式设计
- ✅ 683 行代码

**使用方式**：
```bash
# 在浏览器中打开
start uds-diagnostic.html

# 或直接双击文件
```

**交互说明**：
- 键盘方向键 ← → 翻页
- 鼠标滚轮翻页
- 右上角显示当前页码
- 底部显示操作提示

## 如何生成新示例

参考 `ATAG_PROMPTS.md` 中的模板，使用 AI 生成：

```
请根据 ATAG_PROMPTS.md 中的 [主题] 模板生成网页：

【内容】
[你的技术文档]

【要求】
- 使用电动蓝配色
- 包含 6-8 页
- 代码 600+ 行
- 保存到：./examples/[文件名].html
```
