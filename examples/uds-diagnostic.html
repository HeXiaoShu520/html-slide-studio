协议概述

应用层协议
基于 OSI 第7层，运行于 CAN、LIN、Ethernet 等物理层之上
请求-响应模式
Tester 发送请求，ECU 返回正响应（0x40+SID）或负响应（0x7F）
安全访问
Seed-Key 机制保护敏感操作，不同安全级别对应不同权限
广泛应用
ECU 编程、故障诊断、数据读取，覆盖整车生命周期

核心服务分类

0x10
诊断会话控制
切换 Default / Extended / Programming 会话
0x27
安全访问
Seed-Key 解锁，保护敏感操作
0x22
读取数据
按 DID 读取 ECU 数据标识符
0x19
读取故障码
读取 / 清除 DTC 故障诊断码
0x34/36/37
数据传输
请求下载、传输数据、退出传输
0x11
ECU 复位
硬复位 / 软复位 / 关闭电源

通信流程

1
切换诊断会话
0x10 03 → 切换至 Extended Session
2
请求 Seed
0x27 01 → ECU 返回 0x67 01 [Seed]
3
发送 Key
0x27 02 [Key] → ECU 返回 0x67 02（解锁成功）
4
执行诊断操作
读取数据 / 写入参数 / 下载固件
5
ECU 复位
0x11 01 硬复位，结束诊断会话
