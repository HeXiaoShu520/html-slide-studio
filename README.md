# ATAG Studio
## 车载技术演示文档可视化编辑器

> 专为汽车技术教学设计的 HTML 演示文档生成与可视化编辑工具
> AI 自动生成 + 拖拽可视化编辑 + 一键导出

---

## 项目简介

ATAG Studio 是一个基于浏览器的可视化编辑器，帮助你将汽车技术内容（UDS、SOME/IP、DoIP 等）快速生成为专业的 HTML 演示文档，并提供完整的可视化编辑能力。

**核心工作流：**
```
选择主题风格 → AI 生成内容 → 可视化编辑 → 导出 HTML
```

---

## 功能特性

### AI 生成
- 内置 5 个车载技术场景模板（UDS、SOME/IP、DoIP、相机标定、OTA）
- 支持自定义主题生成
- 兼容 OpenAI 接口格式（可接入本地模型）

### 可视化编辑
- 拖拽组件到画布（信息卡片、流程图、时间线、代码块等）
- 双击直接编辑文字
- 右侧面板调整样式（尺寸、排版、背景、边框、布局）
- 撤销/重做

### 主题系统
- 电动蓝（#0066FF）
- 机械灰（#607D8B）
- 新能源绿（#00BFA5）

### 演示与导出
- 演示模式：全屏翻页播放，键盘/按钮控制
- 预览：新窗口预览完整效果
- 导出 HTML：独立文件，可直接分享
- 另存为 HTML：带元数据，可重新导入编辑
- 自动保存：每 5 秒保存到浏览器本地

## 快速开始

### 环境要求
- Node.js 18+
- npm 或 pnpm

### 启动开发服务器

```bash
cd atag-studio
npm install
npm run dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
```

## 项目结构

```
atag-studio/                  # 主应用（React + Vite）
├── src/
│   ├── components/
│   │   ├── TopBar.tsx        # 顶部工具栏（保存/导出/主题/设备）

│   ├── store/
│   │   └── useAppStore.ts    # 全局状态 + 保存/加载逻辑
│   └── themes/
│       └── themeCSS.ts       # 主题 CSS 变量生成
└── package.json
```

## 文件格式

| 文件 | 用途 |
|---|---|
| `*.html`（另存为） | 工程文件，可重新导入编辑，也可直接在浏览器打开 |
| `*.html`（导出） | 成品文件，包含完整主题样式，用于分享/部署 |

工程 HTML 文件头部包含 `<!-- atag-meta: {...} -->` 注释，存储主题和项目名，导入时自动识别。
