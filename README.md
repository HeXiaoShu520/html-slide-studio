# HTML Slide Studio

> 通用 HTML 演示文档生成与可视化编辑工具

基于 AI 生成 + GrapesJS 可视化编辑，支持多类别演示文档的创建、编辑和导出。

---

## 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:5173

---

## 功能概览

| 功能 | 说明 |
|------|------|
| AI 生成 | 选择类别模板，粘贴参考内容，一键生成演示文档 |
| 可视化编辑 | 拖拽组件、双击编辑文字、右侧面板调整样式 |
| 主题切换 | 电动蓝 / 机械灰 / 新能源绿 |
| 预览 / 演示 | 窗口内全屏预览，演示模式支持键盘翻页 |
| 自动保存 | 每 5 秒保存到 localStorage，重启自动恢复 |
| 导入 / 导出 | 导出 HTML 文件可重新导入继续编辑 |

## 使用指南

### 1. AI 生成文档

1. 点击顶栏 **AI 生成**（星星图标）
2. 配置 API（展开"API 配置"，填入 Endpoint / Key / 模型名）
3. 选择**文档类别**和**模板**
4. 可选：粘贴参考内容（技术文档、Markdown 等）
5. 点击**开始生成**

支持的类别：
- **车载技术**：UDS 诊断、SOME/IP、DoIP、OTA 升级
- **技术培训**：技术方案、系统架构、操作手册
- **产品展示**：产品发布、项目汇报
- **自定义**：自由输入生成指令

### 2. 可视化编辑

- **拖拽组件**：从左侧"组件"面板拖入画布
- **选中元素**：单击选中，双击编辑文字
- **调整样式**：右侧"样式"面板修改尺寸、排版、背景、边框
- **图层管理**：左侧"图层"面板查看和选择元素层级
- **撤销/重做**：顶栏 ↩ ↪ 按钮，或 Ctrl+Z / Ctrl+Y

### 3. 主题切换

顶栏右侧下拉菜单选择主题，实时更新画布配色。

### 4. 预览与演示

- **预览**：点击 👁 图标，在当前窗口内查看完整效果
- **演示模式**：点击 ▶ 图标，进入全屏翻页演示
  - 键盘：↑↓ 方向键 / 空格翻页
  - ESC 关闭演示

### 5. 保存与导出

- **自动保存**：每 5 秒自动保存，无需手动操作
- **手动保存**：点击 💾 图标立即保存
- **导出项目**：点击 📥 图标，导出 `.html` 文件（含元数据，可重新导入）
- **导出 HTML**：点击"导出 HTML"按钮，导出完整可分享的 HTML 文件
- **打开项目**：点击 📂 图标，导入之前导出的 `.html` 项目文件
