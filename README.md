# ATAG - Automotive Technical Animation Generator
## 车载技术动画生成系统

> 专为 AUTOSAR 技术教学设计的网页动画生成方法论
> 将技术文档快速转换为交互式演示网页
> **完全原创设计，可商用**

---

## 📋 项目简介

ATAG 是一套完整的方法论系统，帮助你将 AUTOSAR 相关技术文档（UDS诊断、SOME/IP通信、DoIP升级、相机标定等）快速转换为专业的交互式网页演示。

### 适用场景
- 🎓 **内部培训**：新员工技术培训、知识分享
- 👥 **客户展示**：技术方案演示、产品介绍
- 📚 **文档可视化**：维护手册、操作指南
- 💻 **代码讲解**：代码审查、架构说明

### 核心优势
- ⚡ **快速生成**：5-10分钟从文档到网页
- 🎨 **风格统一**：专业的汽车科技风格
- 🔧 **易于定制**：灵活的配色和布局
- 📱 **响应式设计**：支持桌面和平板
- 💼 **可商用**：完全原创，无版权问题

## 🚀 快速开始

### 5分钟生成你的第一个网页

**步骤 1：准备内容**
```markdown
# UDS 统一诊断服务
UDS 是基于 ISO 14229 的车载诊断协议...
```

**步骤 2：选择模板**
打开 `ATAG_PROMPTS.md`，找到 "UDS 诊断服务" 模板

**步骤 3：发送给 AI**
```
请根据 ATAG_PROMPTS.md 中的 UDS 模板，
使用以下内容生成教学网页：

[粘贴你的内容]

保存到：./output/uds-tutorial.html
```

**步骤 4：在浏览器中打开**
生成的 HTML 文件可直接在浏览器中打开使用

## ❓ 常见问题

### Q: 需要什么技术背景？
**A:** 不需要编程基础，会使用 AI 工具即可。如需深度定制，了解 HTML/CSS 会更好。

### Q: 生成的网页可以商用吗？
**A:** 可以！ATAG 是完全原创设计，无版权问题。

### Q: 支持哪些 AI 工具？
**A:** Claude、GPT-4、文心一言、通义千问等主流 AI 都支持。

### Q: 如何确保风格统一？
**A:** 使用 ATAG_PROMPTS.md 中的标准模板，所有生成的网页会自动保持一致风格。

### Q: 可以导出为 PDF 吗？
**A:** 可以，使用浏览器的"打印为 PDF"功能即可。

### Q: 如何添加公司 Logo？
**A:** 在生成时指定：
```
请在封面页右上角添加公司 Logo
图片路径：./assets/logo.png
尺寸：120x40px
```
