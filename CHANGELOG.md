# HTML Slide Studio 更新日志

## v2.1 - 2026-04-28

### 新增
- 预览/演示模式改为在当前窗口内覆盖显示，不再新开标签页
- 自动保存：每 5 秒保存到 localStorage，启动时自动恢复上次项目
- 导入/导出 HTML 项目文件（含元数据注释，可重新导入继续编辑）
- AI 生成面板支持多类别模板：车载技术 / 技术培训 / 产品展示 / 自定义
- TopBar 新增打开项目、导出项目按钮

### 变更
- 项目定位从车载专用工具升级为通用 HTML 演示文档编辑器
- AI system prompt 改为通用演示文档生成器
- 清理 v1 遗留文档（ATAG_GUIDE.md、ATAG_METHODOLOGY.md 等）
- README 重写，反映新定位


## v2.0 - 2026-04-27

### 重大升级：从方法论工具集升级为可视化编辑器应用

#### 新增：atag-studio 应用
- React 18 + TypeScript + Vite 应用框架
- GrapesJS 可视化编辑器集成
- 三栏 IDE 布局（组件库 / 画布 / 属性面板）

#### 核心功能
- AI 生成：内置 5 个车载技术场景模板，支持 OpenAI 兼容接口
- 可视化编辑：拖拽组件、双击编辑文字、右侧样式面板
- 主题系统：电动蓝 / 机械灰 / 新能源绿，实时切换
- 演示模式：全屏翻页播放，键盘控制
- 设备预览：桌面 / 平板 / 手机响应式预览

#### 保存与导出
- 自动保存：每 5 秒保存到 localStorage，启动自动恢复
- 另存为 HTML：带 `atag-meta` 元数据注释，可重新导入编辑
- 打开 HTML 项目：解析元数据，恢复内容、样式和主题
- 导出 HTML：完整独立文件，内联主题 CSS，可直接分享

#### 组件库
- 信息卡片、卡片网格、流程图、时间线、代码块、数据包表格、层级架构图
